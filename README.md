# exp1 - 任务+积分工具

一个用于激励用户培养习惯的游戏化任务管理应用。

## 功能特性

- 📝 **任务管理** - 创建重复性任务，支持按日/周/月循环
- 📅 **日历视图** - 直观查看任务完成情况
- 🎁 **奖励系统** - 使用积分兑换自定义奖励
- 🎒 **背包系统** - 管理已兑换的奖励物品
- 📊 **数据统计** - 追踪任务完成进度和积分历史
- 💾 **数据备份** - 支持 JSON 格式导入/导出数据

## 技术栈

- **框架**: tauri+react
- **路由**: react-router
- **数据库**: dexie.js(indexeddb)
- **状态管理**: Zustand
- **语言**: TypeScript
- 图标: lucide

## 开发命令

```bash
# 默认启动的服务ip不对, 需要手动指定
bun run tauri android dev -- --host <本机ip>
```

### 视觉风格

- **整体风格**：极简主义，深色主题
- **主色调**：深灰黑 `#1a1a1a`（文字）、红色 `#f56565`（强调）
- **背景色**：深灰 `#1b1b1f`（主背景）、浅灰 `#202127`（卡片背景）
- **字体**：系统默认字体栈
- **图标**：Lucide 线性图标，1.5px 细线条
- **圆角**：小元素 6px，大元素 12px
